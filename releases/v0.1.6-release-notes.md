## Downloads

Download the appropriate installer for your platform:

### macOS (.dmg installers)
- **Apple Silicon (M1/M2/M3/M4)**: `yume-0.1.6-macos-arm64.dmg`
- **Intel x64**: `yume-0.1.6-macos-x64.dmg`

### Windows
- **x64**: `yume_0.1.6_x64-setup.exe`

## Installation

**macOS users:**
1. Download the appropriate .dmg file for your Mac
2. Open the .dmg and drag yume to Applications
3. If blocked by Gatekeeper, right-click and select "Open" or run:
   ```bash
   xattr -cr /Applications/yume.app
   ```

**Windows users:**
1. Download the .exe installer
2. Run it

## Requirements

- macOS 10.13+ (both Apple Silicon and Intel)
- Windows 10/11 x64
- Claude CLI installed and configured (`claude auth login`)
