# yume v0.1.2

## Update Indicator and Clickable Status Badges

This release adds an update indicator and makes status badges clickable.

### What's New

**Update Indicator**
- Visual indicator when updates are available
- Easy access to download new versions
- Checks https://aofp.github.io/yume/version.txt for latest version

**UI Improvements**
- Clickable status badges
- Better interactivity throughout the interface

## What is yume?

yume is a native GUI for Claude Code with:
- 5 specialized core agents (Architect, Explorer, Implementer, Guardian, Specialist)
- Plugin system for custom commands, agents, hooks, skills, and MCP servers
- Background agent execution with git branch isolation
- Persistent memory system via MCP
- Real-time token tracking with cost analytics
- Context compaction with intelligent thresholds
- 12 built-in OLED-optimized themes
- Voice dictation support
- Crash recovery

## Requirements

- macOS 10.13+ (both Apple Silicon and Intel)
- Windows 10/11 x64
- Claude CLI installed and configured (`claude auth login`)
