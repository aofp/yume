# yume v0.1.7

## Fixed macOS Installation

This release introduces .pkg installers for macOS that automatically handle quarantine flags, eliminating the need for manual terminal commands.

### What's Fixed

**macOS Quarantine Issues**
- New .pkg installers automatically remove quarantine flags
- No more "app is damaged" errors on macOS
- No need to run `xattr -cr` commands manually
- Double-click and install - it just works

### What's New

**Installer Format Change (macOS)**
- Switched from .dmg to .pkg format
- .pkg installers provide better installation experience
- Automatic quarantine flag removal built into the installer

## Downloads

Download the appropriate installer for your platform:

### macOS (.pkg installers - auto-remove quarantine)
- **Apple Silicon (M1/M2/M3/M4)**: `yume_0.1.7_arm64.pkg`
- **Intel x64**: `yume_0.1.7_x64.pkg`

### Windows
- **x64**: `yume_0.1.7_x64-setup.exe`

## Installation

**macOS users:**
1. Download the appropriate .pkg file for your Mac
2. Double-click to install
3. That's it! No terminal commands needed.

**Windows users:**
1. Download the .exe installer
2. Run it

## Requirements

- macOS 10.13+ (both Apple Silicon and Intel)
- Windows 10/11 x64
- Claude CLI installed and configured (`claude auth login`)
