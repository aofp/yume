# yume v0.1.0

## Initial Release

The first public release of yume - a native GUI for Claude Code (formerly Claude CLI).

### Features

**5 Specialized Core Agents**
- **Architect** - Plans architecture, decomposes tasks, identifies dependencies
- **Explorer** - Searches files, reads code, understands codebase structure
- **Implementer** - Makes small, focused edits for planned changes
- **Guardian** - Reviews for bugs, security issues, performance problems
- **Specialist** - General-purpose agent for complex tasks

**Plugin System**
- Custom commands
- Custom agents
- Hooks (pre/post actions)
- Skills
- MCP server integration

**Background Agent Execution**
- Run agents in the background
- Git branch isolation for parallel work
- Continue working while agents process

**Persistent Memory System**
- MCP-based memory storage
- Context preservation across sessions
- Knowledge graph integration

**Real-Time Token Tracking**
- Live token usage display
- Cost analytics
- Usage history

**Context Compaction**
- Intelligent thresholds
- Automatic context management
- Preserves important information

**12 Built-in OLED-Optimized Themes**
- Pure black backgrounds for OLED displays
- Variety of color schemes
- 570+ themes available

**Voice Dictation Support**
- Speak your prompts
- Voice-to-text integration

**Crash Recovery**
- Automatic session recovery
- No lost work on crashes

## Downloads

Download the appropriate installer for your platform from the releases page.

## Requirements

- macOS 10.13+ (both Apple Silicon and Intel)
- Windows 10/11 x64
- Claude CLI installed and configured (`claude auth login`)

## Getting Started

1. Install Claude CLI: `npm install -g @anthropic-ai/claude-code`
2. Authenticate: `claude auth login`
3. Download and install yume for your platform
4. Launch yume and start coding with Claude!
