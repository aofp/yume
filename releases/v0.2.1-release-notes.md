# yume v0.2.1

## Maintenance Release

This is a maintenance release with minor improvements and stability fixes.

### What's Included

- Build system refinements
- Minor bug fixes and stability improvements

## Downloads

Download the appropriate installer for your platform:

### macOS (Recommended: .pkg installers)
- **Apple Silicon (M1/M2/M3/M4)**: `yume_0.2.1_arm64.pkg`
- **Intel x64**: `yume_0.2.1_x64.pkg`

### Windows
- **x64**: `yume_0.2.1_x64-setup.exe`

## Installation

**macOS users:**
1. Download the appropriate .pkg file for your Mac
2. Double-click to install
3. The installer automatically removes quarantine flags

**Windows users:**
1. Download the .exe installer
2. Run it

## Requirements

- macOS 10.13+ (both Apple Silicon and Intel)
- Windows 10/11 x64
- Claude CLI installed and configured (`claude auth login`)
