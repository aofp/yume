# yume v0.2.0

## Font Bundling Fix for Windows

This release fixes font loading issues on Windows and improves the macOS installation experience with proper .pkg installers.

### What's Fixed

**Font Bundling for Windows**
- Agave font now properly embedded in Windows builds
- No longer requires manual font installation on Windows
- Consistent typography across all platforms

**macOS Installation**
- Switched to .pkg installers with `--component` mode
- Prevents relocation issues during installation
- Installers auto-remove quarantine flags

### Technical Details

- macOS .pkg installers use component mode to install to fixed location
- Windows builds now include bundled fonts in assets
- Bundle identifier: `io.github.aofp.yume`

## Downloads

Download the appropriate installer for your platform:

### macOS (Recommended: .pkg installers)
- **Apple Silicon (M1/M2/M3/M4)**: `yume_0.2.0_arm64.pkg`
- **Intel x64**: `yume_0.2.0_x64.pkg`

### Windows
- **x64**: `yume_0.2.0_x64-setup.exe`

## Installation

**macOS users:**
1. Download the appropriate .pkg file for your Mac
2. Double-click to install
3. The installer automatically removes quarantine flags - no manual commands needed!

**Windows users:**
1. Download the .exe installer
2. Run it
3. Agave font is bundled - no separate font installation required

## Requirements

- macOS 10.13+ (both Apple Silicon and Intel)
- Windows 10/11 x64
- Claude CLI installed and configured (`claude auth login`)
