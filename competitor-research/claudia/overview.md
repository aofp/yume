# Claudia

## Overview
- **Developer**: Unknown
- **Type**: GUI wrapper for Claude Code
- **Tech Stack**: Tauri 2 (same as yurucode)
- **Pricing**: Free
- **License**: AGPL (open source)

## Key Features
- desktop application for claude code
- visual interface for claude interactions
- advanced ui features
- chat-based interface
- built with tauri 2

## Strengths
- **truly free**: agpl license, permanently free
- **open source**: agpl ensures freedom
- **same tech stack**: tauri 2 benefits
- **community driven**: open development model
- **no vendor lock-in**: can be forked/modified

## Weaknesses
- **agpl license**: restrictive for commercial use
- **limited info**: less public documentation
- **smaller ecosystem**: fewer integrations
- **maintenance uncertainty**: depends on community
- **feature parity**: may lag behind commercial options

## Target Audience
developers valuing open source, privacy-conscious users, contributors wanting to extend functionality

## Competitive Position
competes on openness and freedom, appeals to developers who prioritize agpl licensing over proprietary alternatives
