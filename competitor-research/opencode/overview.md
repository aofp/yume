# OpenCode

## Overview
- **Developer**: Open source community
- **Type**: CLI-based AI coding agent
- **Pricing**: Free (open source)
- **Platforms**: Cross-platform (terminal-based)
- **Models**: 75+ LLM providers supported

## Key Features
- 75+ llm provider support (openai, google, anthropic, local)
- terminal-native workflow
- supports ollama for local models
- major rewrite for maturity
- no gui, pure cli
- works offline with local models

## Strengths
- **truly free**: open source, no costs beyond api
- **model flexibility**: 75+ providers
- **local models**: ollama support for privacy
- **terminal workflow**: integrates with existing tools
- **no vendor lock-in**: switch providers easily
- **privacy focused**: can run completely offline
- **mature codebase**: recently rewritten

## Weaknesses
- **no gui**: terminal only, steeper learning curve
- **setup complexity**: requires configuration
- **less polished**: compared to commercial tools
- **manual context**: user manages context
- **documentation**: may be lacking
- **smaller team**: slower feature development

## Target Audience
terminal-focused developers, privacy advocates, users wanting model flexibility, cost-conscious developers
